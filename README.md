# Testes com jest

Repositório com exemplo de teste de integração utilizando jest

## Installation

Faça o download do repositório

```bash
git clone https://github.com/otallyto/testes-com-jest.git
```
Navegue até a pasta

```bash
cd testes-com-jest
```

Instale as dependencias 

```bash
npm install || yarn install
```
## Usage

```bash
yarn test || npm test 
```
